Primary Tasks

Gameplay Programming:
- Implemented character controller, and majority of player-facing functionality
- Game state managers, AI and behaviour trees for child NPCs
- Developed modular state machine used in Task and Hazard interactions
- In-game UI implementation

Tech Art:
- Tools and systems design, establishing asset pipeline for level and content implementation
- Custom editor tools so game's 120+ unique Hazards could be fully implemented by level designers without writing code
- Shader work, VFX, and in-engine animation
- Development of cutscene and dialogue tech for story-based Campaign Mode
- Implementation of in-engine cutscenes for Campaign Mode

Narrative Design:
- Narrative design and scriptwriting for game's Campaign Mode

Project Overview

A local multiplayer, cooperative party game about extremely poor parenting. Players take on the role of parents and work together to keep their many misbehaving kids alive and well whilst doing everyday tasks like grocery shopping, going to the beach, or hosting a dinner party.

'Think of the Children' was published by Fellow Traveller and released on PC, MacOS, PS4, Xbox One, and Nintendo Switch.
